# imas-csv

アイドルマスター（アイマス）に関するCSVファイルをまとめたリポジトリです。

## ファイル一覧

| ファイル名 | 内容 |
|-----------|------|
| [idols.csv](idols.csv) | 765プロダクション所属アイドルのプロフィール情報 |

## idols.csv の列定義

| 列名 | 説明 |
|------|------|
| 名前 | アイドルの名前 |
| よみがな | 名前のよみがな |
| 年齢 | 年齢 |
| 誕生日 | 誕生日 |
| 身長 | 身長（cm） |
| 体重 | 体重（kg） |
| B | バスト（cm） |
| W | ウエスト（cm） |
| H | ヒップ（cm） |
| 血液型 | 血液型 |
| 出身地 | 出身地 |
| 担当色 | イメージカラー |
| 声優 | 担当声優 |
| ユニット | 所属ユニット |
| 所属事務所 | 所属事務所 |
